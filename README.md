Customer Churn Prediction 
 --Problem

This project uses data from 7,043 telecom customers in California (Q2 2022).
Each row represents one customer and includes:

Demographic details

Location

Tenure (how long they stayed)

Services they use

Whether they joined, stayed, or churned

We also use ZIP code population data to enhance the analysis.

The goal is to predict whether a customer will churn, stay, or join.

-- Model Accuracy
Model	Accuracy
Random Forest	->78.11%
Logistic Regression ->	78.28%
Naive Bayes ->	36.77%
Decision Tree	-> 77.29%
XGBoost Classifier -> 80.86%

-- Why This Is Useful?

Predicting churn helps companies act before customers leave.
For example:

Customer support teams can contact high-risk customers

The company can offer personalized plans or discounts

Helps improve customer satisfaction and reduce losses
